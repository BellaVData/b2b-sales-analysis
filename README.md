# Анализ данных о продажах (B2B Retail)

## Описание проекта  
Аналитический кейс по анализу данных о продажах в B2B-сегменте (крупные розничные сети).

Цель работы — проанализировать динамику продаж, выявить драйверы роста, сезонность, структуру выручки и концентрацию по клиентам на основе исходного Excel-датасета.

## Выполненные работы  

- Проведена очистка и проверка данных (дубликаты, пропуски, типы данных, корректность дат)
- Рассчитана выручка (Цена × Количество)
- Созданы сводные таблицы по следующим направлениям:
  - Динамика выручки (месяц к месяцу)
  - Структура продаж по категориям товаров
  - Анализ по городам
  - Анализ по заказчикам (ARPA, доля в общей выручке)
- Построены диаграммы и графики для визуализации результатов
- Подготовлена презентация с ключевыми выводами и управленческими инсайтами

## Ключевые аналитические метрики 

- Динамика выручки месяц к месяцу  
- Структура выручки по категориям  
- Средний чек (AOV)  
- Средняя выручка на клиента (ARPA)  
- Анализ концентрации выручки и волатильности  

## Структура репозитория  

- `Diagrams.xlsx` — Excel-файл с расчетами, сводными таблицами и графиками  
- [Открыть презентацию](./Presentation.pdf) — Презентация с выводами и бизнес-инсайтами  

## Основные выводы  

- Рост выручки обеспечивается увеличением объема поставок, а не ростом цены  
- Наблюдаются выраженные сезонные колебания  
- Выручка концентрируется в отдельных категориях товаров  
- Клиентский портфель демонстрирует управляемый уровень концентрационного риска  
- Бизнес характеризуется заметной волатильностью показателей
